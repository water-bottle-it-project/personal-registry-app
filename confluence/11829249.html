Team 55 IT Project : Live/public deployment

Created by Ian Chen, last modified on Sep 28, 2022

(Completed Week 6 // End of Sprint 1)

We are using DigitalOcean to automatically deploy our site from the main branch.

A live version of the site, which is automatically re-built on every push to the main branch in our repo, is available at https://personal-registry-app-zx936.ondigitalocean.app/ .

To get an account, either register an account on the homepage or use 1u@mailinator.com which is a pre-populated account from Test User Accounts

The link for the DigitalOcean console is in Team Tools.

We also use other services - MongoDB and Firebase, to provide other backend services (database, file storage, authentication). The links for their consoles are also in Team Tools. They require a .env.local file, which cannot be stored in our Git repository because they contain private keys.

Upon submission, we will send a copy of our .env.local file to the assessor, though in the real-world we would use a secrets manager to even more securely share secret private keys with others.
