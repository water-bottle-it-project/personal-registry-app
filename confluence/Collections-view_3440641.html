Team 55 IT Project : Collections view

Created by Ian Chen, last modified on Oct 23, 2022

Each collection has a

  • name

  • a description (optional)

  • a colour (predefined choices) (optional)

The collections view lists all the collections that a user has created, and allows creating new collections and editing the name, description, and colour of collections.

Anatomy

  • Form with name of new collection, a description box and a colour picker

    • A clear all button to clear progress

    • Unsaved changes alert

  • Vertical list of collections sorted by name

    • Name

      • background colour is the chose colour

    • Description

    • Memories count

    • An edit button allows inline editing of collection name, description, and colour inline, as well as deleting the collection.

      • Confirm button for delete

Design changes [End of Sprint 2]

  • No gradient

  • New search bar

  • Removed memories count
