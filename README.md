<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/water-bottle-it-project/personal-registry-app
```

Go to the project directory

```bash
  cd personal-registry-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
  or
  nodemon app.js
```
